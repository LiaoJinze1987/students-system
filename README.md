# 学生管理システム（Student Management System）

## 概要
本プロジェクトは、シンプルな学生管理システムです。  
ユーザー登録・ログイン機能をはじめ、学生と教師の日常業務をサポートする基本機能を実装しています。

## 主な機能
- ユーザー登録・ログイン
- 学生の休暇申請
- 教師による休暇承認
- 学校からの全体通知配信
- 学生・教師情報の管理

## 技術スタック
- バックエンド：Spring Boot 3
- フロントエンド：Vue 3 + Element Plus

## 説明
小規模なシステムを想定し、基本的な業務フロー（申請・承認・管理）を一通り実装しています。

# Student Management System

## Overview
This project is a lightweight student management system.  
It implements basic features to support daily operations between students and teachers.

## Features
- User registration and login
- Student leave request
- Teacher approval for leave requests
- Global announcements from the school
- Student and teacher information management

## Tech Stack
- Backend: Spring Boot 3
- Frontend: Vue 3 + Element Plus

## Description
This project simulates a small-scale management system and covers the core workflow, including request, approval, and data management.
